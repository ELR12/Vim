First:  Copy all contents out of Vim folder, and then delete Vim folder, you don't need this folder

On Linux machine:
	- Add .vimrc file to root:  ~/    -->  ~/.vimrc 
	- Add "bundle" file to: ~/.vim/   -->  ~/.vim/bundle/




On Windows machine:
	- Add .vimrc file to:  /c/Program Files/Git/etc/  -->  /c/Program Files/Git/etc/.vimrc
	- Add "bundle" file to: /c/Users/*username*/.vim/ --   /c/Users/*username*/.vim/bundle/
